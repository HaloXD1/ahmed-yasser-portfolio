# Ahmed Yasser Shalaby Portfolio

Personal portfolio for data engineering, analytics engineering, cloud security automation, and business systems simulation work.

Live site: https://ahmedyassershalaby.github.io/ahmed-yasser-portfolio/

Core idea: data systems that make messy work easier to trust.

## Local Preview

```bash
npm install
npm run dev
```

Open the Vite URL printed by the command.

## Deploy

GitHub Pages builds the app from `main` and publishes `dist`.
