# Goal Run Logs

Short logs for recurring Codex career goals.

Naming:

- `YYYY-MM-DD_internship_application_machine.md`
- `YYYY-MM-DD_data_engineering_skill_sprint.md`
- `YYYY-MM-DD_portfolio_proof_upgrade.md`

Each log should include:

- Status.
- Work done.
- Top next actions.
- Blockers, if any.
