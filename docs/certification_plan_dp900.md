# Certification Plan: Microsoft DP-900

Recommendation: **Microsoft Certified: Azure Data Fundamentals (DP-900)**.

Why this one:

- Beginner level.
- Directly supports data/cloud internship positioning.
- Recognized more broadly than small course badges.
- Matches Business Informatics + Data Engineering direction.
- Good signal for databases, analytics workloads, and cloud data basics.

Official pages:

- Certification: https://learn.microsoft.com/en-us/credentials/certifications/azure-data-fundamentals/
- Study guide: https://learn.microsoft.com/en-gb/credentials/certifications/resources/study-guides/dp-900

## Exam Topics

Based on the official study guide:

- Core data concepts: 25-30%.
- Relational data on Azure: 20-25%.
- Non-relational data on Azure: 15-20%.
- Analytics workloads on Azure: 25-30%.

## 10-Day Study Plan

### Day 1

- Read certification overview.
- Learn transactional vs analytical workloads.
- Write 10 flashcards.

### Day 2

- Relational data basics.
- Tables, keys, normalization, SQL, indexes.
- Practice 10 SQL questions.

### Day 3

- Non-relational data basics.
- Document, key-value, graph, column-family.
- Compare MongoDB/Cosmos-style use cases.

### Day 4

- Azure storage basics.
- Blob storage, data lake concepts, file formats.
- Connect to Parquet/DuckDB knowledge.

### Day 5

- Analytics workload basics.
- Warehouses, lakehouses, batch vs streaming.
- Map to Retail and SaaS projects.

### Day 6

- Azure SQL, Synapse/Fabric/Databricks concepts.
- Do Microsoft Learn modules.

### Day 7

- Power BI and visualization concepts.
- Review BI/data model vocabulary.

### Day 8

- Official practice assessment.
- Record weak areas.

### Day 9

- Review weak areas.
- Retake practice assessment.

### Day 10

- Final review.
- Schedule exam only if practice score is consistently strong.
