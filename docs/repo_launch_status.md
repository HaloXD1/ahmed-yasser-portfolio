## GitHub Pins

GitHub profile pinning is a manual UI step. The GitHub GraphQL schema exposes pinned items for reading, but no profile repository pin mutation is available through the current CLI schema.

Manual steps:

1. Open GitHub profile.
2. Click **Customize your pins**.
3. Select:
   - `data-pipeline-kpi-dashboard`
   - `saas-analytics-engineering-pipeline`
   - `cloud-security-posture-auditor`
   - `ahmed-yasser-portfolio`
   - `outlook-calendar-automation`
   - `egx-research`
4. Save pins.
